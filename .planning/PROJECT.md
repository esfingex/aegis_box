# PROJECT.md — aegis_box

## Vision & Scope
Aegis Box is a premium local sandboxing manager and legacy dynamic library isolator tailored for Arch Linux and CachyOS. It enables secure execution of untrusted applications (like browsers, Wine games, or unknown scripts) in RAM-backed ephemeral overlays using Firejail, resolving missing or legacy shared library dependencies on the fly without system corruption.

---

## Architecture & Components

### Core Components:
*   **Database Engine (`src/database.py`)**: Manages the local SQLite database (`~/.config/aegis_box/aegis_box.db`) containing tables for active `sessions`, registered `apps`, legacy `libs_cache`, and global environment `settings`.
*   **CLI Orchestrator (`src/cli.py`)**: Entry point for execution command pipelines (`aegis-box run`), bridge control, and library downloading.
*   **GUI Controller (`src/main.py`)**: Main PySide6 window managing application navigation, window splitting, and asynchronous worker execution threads.
*   **Modular Views (`src/views/`)**:
    *   `sidebar.py`: Sidebar navigation controls.
    *   `workspace.py`: Main container tables (pending sessions, active libraries, settings).
    *   `detail_drawer.py`: Drawer panels for profile tweaking, library management, and terminal diff audits.
    *   `dialogs.py`: Form modals to register applications and instantiate custom profiles.
*   **Style & Theme (`src/style.py`)**: Premium dark-mode stylesheet rules (QSS) for desktop widgets.
*   **Internationalization (`src/i18n.py`)**: Key-value JSON localization system (under `src/locales/`) supporting runtime English/Spanish translation.

## Core Ecosystem Conventions

1.  **Rust Token Killer (RTK)**: All terminal commands executed by developer agents must be prefixed with `/home/esfingex/.local/bin/rtk`.
2.  **Bilingual Caveman Format (BCF)**: Observations stored in the CaveMem registry must use BCF format: `[EN]` in compressed, article-free style, followed by `[ES]` in human-friendly Spanish.
3.  **Zero-Pollution Encapsulation**: Keep local configurations confined inside user home paths (`~/.config/aegis_box/`, `~/.local/share/aegis_box/`).
4.  **No Emojis or Icons**: Planning documents (`PROJECT.md`, `ROADMAP.md`, `STATE.md`, `continue-here.md`) must remain text-only to maximize LLM context efficiency.
